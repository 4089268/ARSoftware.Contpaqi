# ARSoftware.Contpaqi.Comercial.Sql.Models

`ARSoftware.Contpaqi.Comercial.Sql.Models` contiene las clases que representan el esquema de las tablas de las bases de datos de Contpaqi Comercial.

Este proyecto es utilizado principalemente por [ARSoftware.Contpaqi.Comercial.Sql](https://www.nuget.org/packages/ARSoftware.Contpaqi.Comercial.Sql).

## Instalación

Instala este paquete con [NuGet](https://www.nuget.org/packages/ARSoftware.Contpaqi.Comercial.Sql.Models):

```dotnetcli
dotnet add package ARSoftware.Contpaqi.Comercial.Sql.Models
```
