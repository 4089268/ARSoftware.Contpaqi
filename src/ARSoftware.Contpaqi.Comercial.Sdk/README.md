# ARSoftware.Contpaqi.Comercial.Sdk

`ARSoftware.Contpaqi.Comercial.Sdk` tiene declaradas todas las funciones y clases utilizadas por el SDK de CONTPAQi Comercial y CONTPAQi Factura Electronica.

Usa este proyecto si solamente necesitas las funciones y clases del SDK declaradas para tu manualmente implementar los servicios para crear, modificar, eliminar, y consultar los catalogos.

Este proyecto es utilizado principalmente por [ARSoftware.Contpaqi.Comercial.Sdk.Extras](https://www.nuget.org/packages/ARSoftware.Contpaqi.Comercial.Sdk.Extras/) el cual implementa varios repositorios y servicios para facilmente crear, modificar, eliminar y consultar catalogos.

## Instalación

Instala este paquete con [NuGet](https://www.nuget.org/packages/ARSoftware.Contpaqi.Comercial.Sdk.Extras/):

```dotnetcli
dotnet add package ARSoftware.Contpaqi.Comercial.Sdk.Extras
```

Puedes explorar el proyecto [Sdk.ConsoleApp](https://github.com/AndresRamos/ARSoftware.Contpaqi.Comercial/tree/master/samples/Sdk.ConsoleApp) para ver ejemplos de como utilizar este paquete.
