# ARSoftware.Contpaqi.Comercial.Sdk.Extras

`ARSoftware.Contpaqi.Comercial.Sdk.Extras` extiende la funcionalidad de `ARSoftware.Contpaqi.Comercial.Sdk` para facilitar el desarrollo de aplicaciones que se integren con CONTPAQi Comercial. 

Este proyecto implementa diferentes repositorios y servicios para facilmente iniciar una conexión con Contpaqi Comercial y crear, modificar, eliminar, y consultar los diferentes catalogos catalogos.

## Instalación

Instala este paquete desde [NuGet](https://www.nuget.org/packages/ARSoftware.Contpaqi.Comercial.Sdk.Extras/).

```dotnetcli
dotnet add package ARSoftware.Contpaqi.Comercial.Sdk.Extras
```

## Ejemplos

Puedes explorar el proyecto [Sdk.Extras.ConsoleApp](https://github.com/AndresRamos/ARSoftware.Contpaqi.Comercial/tree/master/samples/Sdk.Extras.ConsoleApp) para ver ejemplos de como utilizar este paquete.
