# ARSoftware.Contpaqi.Comercial.Sql

`ARSoftware.Contpaqi.Comercial.Sql` te permite acceder y consultar las bases de datos de Contpaqi Comercial desde .NET utilizando Entity Framework Core.

Este proyecto implementa los DbContexts para consultar la bases de datos Generales y Empresas y tambien implementa diferentes repositorios para consultar facilmente los diferentes catalogos.

## Instalación

Instala este paquete con [NuGet](https://www.nuget.org/packages/ARSoftware.Contpaqi.Comercial.Sql):

```dotnetcli
dotnet add package ARSoftware.Contpaqi.Comercial.Sql
```

## Ejemplos

Puedes explorar el proyecto [Sql.ConsoleApp](https://github.com/AndresRamos/ARSoftware.Contpaqi.Comercial/tree/master/samples/Sql.ConsoleApp) para ver ejemplos de como utilizar este paquete.
